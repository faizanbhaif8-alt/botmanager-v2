# Bot Manager V2 - AI-Powered Bot Factory 🤖

## Overview
Bot Manager V2 is a production-ready web application that generates complete bot projects from natural language prompts and deploys them directly to GitHub. Powered by OpenRouter AI with DeepSeek fallback, it creates professional, responsive web applications on demand.

## Features
- 🎯 **AI-Powered Generation**: Create bots from natural language descriptions
- 🚀 **One-Click Deployment**: Push generated code directly to GitHub
- 🎨 **Modern UI**: Cinematic chat interface with Tailwind CSS
- 🔄 **Fallback Logic**: Automatic API fallback (OpenRouter → DeepSeek → Local)
- 📱 **Responsive Design**: Mobile-first, fully responsive layouts
- 🔒 **Production Ready**: Error handling, logging, and security best practices

## Tech Stack
- **Backend**: Flask (Python 3.9+)
- **Frontend**: HTML5, Tailwind CSS, Vanilla JS
- **APIs**: OpenRouter, DeepSeek, GitHub API
- **Automation**: Playwright (browser automation)
- **Media**: MoviePy (video/audio processing)
